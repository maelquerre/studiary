# Studiary

School management system.

## Getting started

### Prerequisites

- PHP 7.3.9 or higher
- [Symfony CLI](https://symfony.com/download)

### Installation

1. Clone the repo
```sh
git clone https://github.com/maelquerre/studiary
```
2. Install the dependencies
```sh
composer install
```

## Develop

Start the local Symfony web server
```sh
 symfony server:start
```
